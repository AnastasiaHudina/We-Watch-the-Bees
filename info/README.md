# Справочные материалы проекта

Здесь лежит всё, что не относится к коду приложения: документация, заметки для разработки и архив.

## Структура

| Папка | Содержимое |
|-------|------------|
| `docs/` | PDF по проекту, инструкция по установке (`install.md`) |
| `dev/` | Быстрый запуск (`quick-start.txt`), учётные данные суперпользователя (`superuser.txt`) |
| `archive/` | Устаревшие черновики и прототипы |

## Приложение

Рабочий код — в `WeWatchTheBees/`:

- `backend/` — Django API
- `frontend/` — React (Vite)
- `emulators/` — эмуляторы датчиков
